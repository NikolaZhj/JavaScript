## 作品简介（简述作品创意、表达形式及特色等，限500字）

本作品以中国传统"飞花令"诗词游戏为文化内核，以中文数字（一至万）为令数，融合横版探索寻访与回合制对战玩法，构建了一款"游山玩水觅诗仙"的像素风 H5 游戏。

**创意来源**：飞花令源于唐代文人雅集，是中华诗词文化的活态遗产。作品将"以诗会友"的传统雅趣转化为数字互动体验，让玩家在 5000 像素宽的横版山水世界中自由探索，邂逅李白（诗仙）、杜甫（诗圣）、苏轼（东坡居士）、王维（诗佛）四位大诗人，通过飞花令对战一决高下。

**表达形式**：采用纯 HTML5 Canvas 像素渲染技术，不使用任何图片资源，仅通过代码绘制王国新大陆美学的像素世界——包含蓝天渐变、三层视差山峦、多层草地、石径小路、动态河流、凉亭石桥、柳树竹林等丰富场景元素；角色为像素块风格，配合走路动画、气泡对话、8bit 音效、粒子爆散特效、屏幕震动等，营造沉浸式游戏体验。

**核心特色**：
1. **双模式设计**——探索冒险（走近诗人触发 Boss 战）与双人对战（PvP）自由切换；
2. **文化科普**——内置 180+ 条成语和 48 条经典诗句数据库，词汇即武器，在对战中自然记忆传统文化；
3. **信息可视化**——HP 血条、20 秒倒计时、气泡展示、小地图导航、胜率统计等；
4. **低门槛提示**——💡 提示按钮自动帮玩家从词库中随机选句出招；
5. **跨平台运行**——单文件 HTML，双击浏览器即玩，PC 和手机均可。

## 简述作品制作过程及技术要点（限300字）

**制作过程**：
1. 确定"数字飞花令"主题，构建成语（180+）与诗句（48）两大数据库，以 Map 结构存储实现 O(1) 快速查询；
2. 用 HTML5 Canvas 绘制王国新大陆风格像素场景，包括天空渐变、三层视差山峦、多层草地、动态河流等元素；
3. 实现横版探索系统——键盘移动角色、相机跟随、Boss 碰撞检测触发战斗；
4. 开发回合制对战系统——令数抽取、词汇校验（Set 去重）、20 秒倒计时、三局两胜制、Boss AI 自动出招；
5. 集成音效（Web Audio API 合成 8bit 音效）、粒子特效、屏幕震动、气泡对话等增强体验；
6. 开发 PvP 双人对战模式与模式切换状态机。

**技术要点**：
- 纯原生 JavaScript + Canvas API，零框架依赖
- `requestAnimationFrame` 驱动 60fps 游戏循环，deltaTime 帧无关动画
- ES6 Map/Set 数据结构实现词库管理与去重
- `ctx.fillRect` 像素块绘制，`imageSmoothingEnabled = false`
- Web Audio API OscillatorNode 合成 Square 波型音效
- 三层视差滚动（0.2/0.4/0.7 倍速）增强纵深感
- CSS3 @keyframes 动画处理消息弹出
